Housing Projects API

This section was made for developers use the API to get or post housing projects from Atlántico-Colombia. Type of request used mainly for suggestions (based on incomplete names). Users just can make request if registered and logged section to enable the section of "Generate API Key" and get the api_key, be aware that api_key is valid for 3 months, after that, you have to refresh on same section. To go the api service in swagger, you can go in next link.

Link API_REST: Api house-finder swagger

Servers

The search route will search the project database for a project that matches the city queried. Use the loc parameter to pass a location name. 


              # Here is a curl example
              import requests

              url = "http://localhost:8000/api/v1/location?loc={location}"

              headers = {
                'api_key': '{public_apikey}'
              }
              
              response = requests.request("GET", url, headers=headers)

              print(response.text)
