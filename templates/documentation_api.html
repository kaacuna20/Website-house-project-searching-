Housing Projects API

This section was made for developers use the API to get or post housing projects from Atlántico-Colombia. Type of request used mainly for suggestions (based on incomplete names). Users just can make request if registered and logged section to enable the section of "Generate API Key" and get the api_key, be aware that api_key is valid for 3 months, after that, you have to refresh on same section.

Servers

The search route will search the project database for a project that matches the city queried. Use the loc parameter to pass a location name. 


              # Here is a curl example
              import requests

              url = "http://127.0.0.1:5002/location?api_key={apikey}&loc={location}"

              response = requests.request("GET", url)

              print(response.text)

To get characters you need to make a GET call to the following url, for example:
user search by location "Ciudad mallorquin": http://127.0.0.1:5002/location?api_key=UserSecretApikey&loc=ciudad mallorquin

Update the price of sale department at a particular project due to price change every year or during current year. Using the id and new_price parameters. This request just is allowed by administration user. 


              # Here is a curl example
               import requests

                url = "http://127.0.0.1:5002/update-price/?api_key={api_key}&new_price={new_price}&project_id={project_id}"

                response = requests.request("PATCH", url, headers=headers, data=payload)

                print(response.text)

To get characters you need to make to PATH call to the following url, for example :
administrator user patch new_price=350.020.000 - 556.960.000, the endpoint is:
http://127.0.0.1:5002/update-price/?api_key=TopSecretAPIKey&new_price=350.020.000 - 556.960.000&project_id=2

Adds a new project entry to the database. Requires authentication with api-key parameter.


              # Here is a curl example
                import requests

                url = "http://127.0.0.1:5002/add?api_key={api_key}&name={name}&logo={logo}&location={location}&city={city}&company={company}
                                &address={address}&url_map={url_map}&address_sale={address_sale}&contact={contact}&stratum={stratum}
                                &area={area}&price={price}&type={type}&description={description} &url_website={url_website}"


                response = requests.request("POST", url, headers=headers, data=payload)

                print(response.text)

QUERY PARAMETERS

# Field Type Description
1 api_key string your API key
2 name string name of project
3 logo string url_img logo of project
4 location string location of project
5 city string city from project
6 company string company built project
7 address string address of project
8 url_map string url of ubication of project (Google Map)
9 address_sale string address of sales point
10 contact string number contact with company
11 stratum string stratum of sector
12 area string Area space of department in m2
13 price string current cost of department
14 type string whether project is VIS or NO VIS
15 url_img string image of background of set of buildings
16 description string summary about the project
17 url_website string url of website of project

Deletes a project from the database. You will need to provide the id of the project to delete as a route. This request just is allowed by administration user.


              # Here is a curl example
        import requests

        url = "http://127.0.0.1:5002/project-closed?api_key={api_key}&project_id={project_id}"

        response = requests.request("DELETE", url, headers=headers, data=payload)

        print(response.text)

To get characters you need to make a DELETE call to the following url, for example:
administrator user delete project with id_project=5, the endpoint is: http://127.0.0.1:5002/project-closed?api_key=TopSecretAPIKey&project_id=5
